# Usage
<!-- usage -->
```sh-session
$ npm install -g syn
$ syn COMMAND
running command...
$ syn (-v|--version|version)
syn/0.0.0 linux-x64 node-v10.16.3
$ syn --help [COMMAND]
USAGE
  $ syn COMMAND
...
```
